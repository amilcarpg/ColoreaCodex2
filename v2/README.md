# PaintMe.club Home Pack for Codex

Este paquete deja listo el material para pedirle a Codex la implementacion de una nueva home para `paintme.club` basada en el mockup generado.

## Archivos

- `paintme-home-context.md`: contexto corto del problema y objetivos.
- `paintme-home-backlog.md`: backlog priorizado `P1/P2/P3` para desarrollo.
- `paintme-home-prompt.md`: prompt principal de implementacion con criterio de fidelidad visual.
- `paintme-home-runbook.md`: instrucciones de uso para lanzar la tarea en Codex.
