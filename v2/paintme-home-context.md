# Contexto

La home actual de `https://www.paintme.club/` funciona, pero transmite mas una demo o MVP que un producto terminado. El objetivo es rediseñarla para que se vea como una landing real lista para producir, manteniendo foco en familias y niños.

## Metas

- Clarificar la propuesta de valor en el primer pantallazo.
- Mejorar branding, jerarquia visual y confianza.
- Hacer mas facil descubrir categorias y dibujos.
- Reforzar la utilidad para padres y familias.
- Mantener una experiencia muy buena en movil.
- Aproximar la implementacion lo mas posible al mockup visual de referencia.

## Restricciones

- No convertir la web en una landing SaaS generica.
- No simplificar el hero.
- No quitar secciones del mockup.
- Priorizar fidelidad visual, con ajustes minimos por responsive, accesibilidad o stack.
